Design choices:
• While the project was introduced in class, I had an idea to include a map of the world in my project, so as to provide the drop down functionality in a more user-friendly and interactive manner. This idea was encouraged further when we saw the sample video attached in the link on the assignment details. The video has a color-coded map on the top right corner of the video.
• After looking up the map online, I found an interactive map on citation: https://www.jasondavies.com/. Including the map gives the user another way to pick the country for the data to be displayed.
• Next, I designed the line graph in such a way that it animates the line to appear more visually appealing.
• While adding the comparison feature as well, I included the map feature and provided the user to pick the second country.
• Since looking up small countries might be difficult to look up on the map, I have provided a drop down menu on the page as well. Color coding was chosen such that some color-blind people can view it properly.
• I used the radio buttons to toggle between a bargraph, linegraph and a scatterplot.
• In my scatterplot, I created a simple scatterplot of countries, along with a drop down on x axis, y axis and for the radius of the circles representing the countries.
• Using the play button user can animate the scatterplot over the years
• I plotted the continents on the side, giving the user a way to just view the countries of a specific continent, since the other countries fade out.
• Using the radio buttons the user can toggle between the scatterplot and the bar/line graph page.
